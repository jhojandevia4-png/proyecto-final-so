Proyecto Final Sistemas Operativos - Grupo 8

Este proyecto implementa una infraestructura basada en Docker y Kubernetes utilizando una máquina virtual Xubuntu 24.04.

Objetivos

  • Implementar contenedores Docker.
  • Orquestar servicios mediante Kubernetes.
  • Publicar una aplicación web usando Nginx.
  • Gestionar versiones mediante Git y GitHub.

Tecnologías Utilizadas

  • Xubuntu 24.04 LTS
  • Docker Engine
  • Docker Compose
  • Kubernetes
  • Minikube
  • Nginx
  • Git y GitHub
