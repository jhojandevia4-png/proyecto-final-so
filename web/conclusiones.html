Conclusiones del Proyecto

La implementación del proyecto permitió comprender el uso de contenedores Docker para el despliegue de aplicaciones.

El uso de Kubernetes facilitó la administración y escalabilidad de los servicios mediante Deployments y Services.

La integración de Git y GitHub permitió gestionar versiones y mantener respaldo del proyecto.

La utilización de máquinas virtuales permitió crear un entorno controlado para las pruebas y configuraciones.

El proyecto fortaleció conocimientos en virtualización, administración de sistemas y tecnologías de nube.
