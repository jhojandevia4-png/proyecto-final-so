Conclusiones del Proyecto

Docker permitió desplegar aplicaciones de manera rápida y organizada mediante contenedores.
Docker Compose facilitó la administración de múltiples servicios desde un único archivo de configuración.
Kubernetes permitió comprender conceptos de despliegue, réplicas y escalabilidad de aplicaciones.
Git y GitHub ayudaron a mantener el control de versiones y el trabajo colaborativo del grupo.
El proyecto fortaleció conocimientos en virtualización, contenedores y administración de sistemas.
