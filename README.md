## Descripción

Proyecto desarrollado para la asignatura Sistemas Operativos.

El objetivo fue implementar una arquitectura basada en contenedores utilizando Docker, Docker Compose y Kubernetes, desplegada sobre una máquina virtual Ubuntu.

## Tecnologías Utilizadas

* Ubuntu 24.04 LTS
* Docker Engine
* Docker Compose
* Kubernetes
* Minikube
* Nginx
* Python
* Git
* GitHub

## Componente 1: Virtualización

Se configuró una máquina virtual Ubuntu con conectividad de red y acceso a Internet para ejecutar los servicios requeridos.

## Componente 2: Docker

Se construyeron dos contenedores:

* Frontend con Nginx
* Backend con Python

Se utilizó Docker Compose para la orquestación de servicios.

## Componente 3: Kubernetes

Se desplegó un Deployment con réplicas utilizando Kubernetes y Minikube.

También se configuró un Service tipo NodePort para exponer la aplicación.
